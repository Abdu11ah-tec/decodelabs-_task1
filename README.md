# Project 1: Data Cleaning & Preparation

## About the Project

This project was completed as part of the DecodeLabs Data Analytics Industrial Training Program. The main goal was to clean a raw dataset and make it ready for analysis. Since real-world data often contains missing values, duplicate records, and inconsistent formatting, the first step was to improve the quality of the data before performing any analysis.

## What I Did

During this project, I carefully examined the dataset and performed several data cleaning tasks, including:

- Checked for missing (null) values.
- Removed duplicate records.
- Standardized text values by removing extra spaces and correcting inconsistencies.
- Fixed date, number, and text formatting issues.
- Filled missing numerical values using the median.
- Filled missing categorical values using the most common value (mode).
- Verified that there were no duplicate IDs.
- Reviewed the dataset to ensure it was clean and consistent.

## Result

After completing the cleaning process, the dataset became more reliable and ready for further analysis. Missing values were handled appropriately, duplicate records were removed, and the data was formatted consistently. This cleaned dataset will be used for the next phase of the project, which is Exploratory Data Analysis (EDA).

## What I Learned

Working on this project helped me understand the importance of data cleaning in the analytics process. I gained practical experience in identifying data quality issues, handling missing values, removing duplicates, and preparing datasets for analysis. It also improved my confidence in using Python, Pandas, and Excel for real-world data preparation tasks.
